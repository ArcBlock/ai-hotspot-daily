# 热点速览 — 写作与设计规范

## 文字约束

### Hero 区域
- **hero.title**: ≤20 个汉字，结论式标题（不是新闻标题）
  - ✅ "春节档票房格局：沈腾仍是喜剧之王"
  - ✅ "油价五连涨：加满一箱多花 15 元"
  - ❌ "春节档电影票房出炉"（太中性）
  - ❌ "国际油价变动情况"（不够具体）

- **hero.takeaway**: ≤30 个汉字，一句话核心洞察
- **hero.keyNumbers**: 刚好 3 个
  - 每个 value ≤6 字（数字为主，如 "3.2亿"、"↑47%"）
  - 每个 label ≤6 字（说明，如 "总票房"、"同比增长"）

### Cards 区域
- **card.title**: ≤15 个汉字
- **card.insight**: ≤20 个汉字，一句话洞察

### 总文字量
整个页面文字总量不超过 200 字（hero + 所有 cards）

## 图表类型匹配规则

### Hero 图表（6 种）

| 数据特征 | 图表类型 | 示例场景 |
|---------|---------|---------|
| 有排名/榜单数据 | bar（横向柱状图）| 票房排行、热搜排名 |
| 有时间序列数据 | line（折线图）| 房价走势、股市波动 |
| 有占比/构成数据 | ring（环形图）| 市场份额、投票结果 |
| 有事件先后顺序 | timeline（时间线）| 突发事件进展、政策演变 |
| 有 A vs B 对比 | versus（左右对比）| 产品对比、球队对决 |
| 有地域分布数据 | heatmap（热力色块）| 各省数据、全球分布 |

### Card 微型图表（6 种）

| 模板 | 适用场景 | 数据格式 |
|------|---------|---------|
| miniBar | 排名/对比 | `{ items: [{label, value, color?}] }` (2-3 条) |
| trend | 涨跌/变化 | `{ direction: "up"|"down", value: "47%", label: "较昨日" }` |
| progress | 完成度/占比 | `{ value: 73, label: "完成度", color?: "#hex" }` |
| bigNumber | 突破性数字 | `{ value: "3.2亿", label: "累计票房" }` |
| versus | A vs B | `{ left: {label, value}, right: {label, value} }` |
| tags | 热议关键词 | `{ items: ["关键词1", "关键词2", ...] }` (3-5 个) |

## 写作风格

- 简洁、直白、没有营销语气
- 不做主观评价，只呈现事实和数据
- 不使用 emoji
- 所有数据必须有来源
- 不用"据悉""据了解"等废话前缀
- 数字用阿拉伯数字，不用中文数字
